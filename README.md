## Git commit 메시지 컨벤션
- **Feat**: 새로운 기능을 추가
- **Test**: 테스트 코드 추가
- **Chore**: 간단한 수정
- **Remove**: 코드 및 파일의 삭제
- **Init**: 프로젝트 초기 생성
- **Comment**: 필요한 주석 추가 및 변경
- **Docs**: 문서작성
